# Archive

This folder stores non-active files moved out of runtime paths to keep the project structure clean.

Subfolders:
- `sql-root-legacy/`: legacy root SQL scripts (manual/one-off history).
- `components/teacher-experiments/`: non-active component variants.
- `types/`: duplicate/unused type files moved from active source paths.
- `assets/`: duplicate assets not used by runtime paths.

Policy:
- Archive first, delete later after validation.
